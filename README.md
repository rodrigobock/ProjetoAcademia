# ProjetoAcademia

Sistema de gestão de academia desenvolvido em C# com interface gráfica (Windows Forms).

## Tecnologias
- Linguagem: C#
- Framework: .NET (Windows Forms)
- Ambiente Recomendado: Visual Studio

## Como Executar
1.  Abra o arquivo de solução (`Academia.sln`) dentro da pasta `Academia` com o **Visual Studio**.
2.  Certifique-se de ter as dependências do .NET instaladas para o projeto.
3.  Clique em "Iniciar" ou "Start" no Visual Studio para compilar e executar a aplicação.
